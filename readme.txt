
「Origin」はWordPressでコーポレートサイトを作る時に便利なスターターキットです。
このテーマから子テーマを作るのではなく、直接カスタマイズすることを想定しています。

「Origin」では“よく使う機能”をあらかじめ追加してある他、
“めったに使わない機能”をオフにすることでWordPressをシンプルに使えるよう設計しています。

* アイキャッチ画像使用
* コメント使用せず
* 言語翻訳なし

* ニュース用カスタム投稿（news）, カスタムタクソノミー（news_cat）を初期登録

* 投稿の自動整形（wpautop, wptexturize）を停止
* カスタムメニューで出力される<div>や余分な id を除去
* メディアの自動生成機能は停止

* ダッシュボード無効化（管理者以外）
